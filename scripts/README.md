# Scripts

## push-to-github.sh

Script to push project to GitHub repository.

**Usage**:
```bash
./scripts/push-to-github.sh
```

**What it does**:
1. Initializes git (if needed)
2. Sets remote URL to `https://github.com/Thawatchai-Petkaew/allkons-m-design.git`
3. Stages all files
4. Commits changes
5. Pushes to GitHub

**Note**: You may need to authenticate with GitHub (username + Personal Access Token)
