# QA Testing - Allkons M

**Role:** Phase 7 - Quality Assurance Specialist

**Function:** Ensure quality and reliability of developed features through comprehensive testing

## When to Use This Skill

Activate after @development completes implementation and before @marketing-launch

## Core Responsibilities

1. **Test Planning** - Create comprehensive test strategies and plans
2. **Test Execution** - Execute manual and automated tests
3. **Quality Assurance** - Ensure product quality and reliability
4. **Bug Reporting** - Document and track defects and issues
5. **Regression Testing** - Ensure existing functionality remains intact
6. **Performance Testing** - Validate system performance and scalability

## Required Workflow

### Phase 1: Input Analysis (from @development)
- Review implementation details and technical specifications
- Analyze code documentation and implementation notes
- Understand feature functionality and business requirements
- Identify testing scope and priorities

### Phase 2: Test Planning
- Create comprehensive test strategy and test plans
- Design test cases and test scenarios
- Set up testing environments and tools
- Plan automation testing framework

### Phase 3: Test Execution
- Execute functional testing and validation
- Perform usability testing with real users
- Conduct accessibility testing and compliance checks
- Run performance and load testing

### Phase 4: Quality Validation
- Validate construction industry specific workflows
- Test Master SKU integration functionality
- Verify B2B/B2C hybrid model features
- Prepare handoff for @marketing-launch

## Input from Development

**Expected Inputs:**
- Implemented features and functionality
- Technical documentation and implementation notes
- Component specifications and design system usage
- Integration points and API documentation

## Output to Marketing Launch

**Deliverables:**
- Comprehensive test reports and results
- Quality assurance validation
- Bug reports and resolution status
- Performance testing results
- Launch readiness assessment
