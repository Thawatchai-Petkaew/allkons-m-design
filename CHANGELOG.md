# Changelog

All notable changes to Allkons M Design will be documented in this file.

## [0.1.0] - 2024

### Added
- Initial MVP release
- Phone OTP Authentication (Supabase + Mock OTP fallback)
- Seller Dashboard (Organization, Shop, Branches)
- Buyer Marketplace (Guest Mode)
- Admin Dashboard
- Prisma Database integration
- Supabase Auth integration
- Design system integration
- Complete documentation

### Changed
- Project name: "Allkons M" → "Allkons M Design"
- Package name: "design-system-app" → "allkons-m-design"

### Technical
- Next.js 15 + React 19 + TypeScript
- Prisma 7 for database management
- Supabase for authentication
- Ant Design 5 for UI components
